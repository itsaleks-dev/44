# React Comtext App
A simple example of using the **React Context API** for global state management without props.

## Demo: 
https://44-hazel.vercel.app

## Install
git clone https://github.com/itsaleks-dev/44
cd my-context-app
npm install
npm run dev

## Technologies used
	•	React + Vite
	•	Context API
	•	useContext / useState
